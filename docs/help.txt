ПРОГРАММА "КАЛЬКУЛЯТОР"
Версия 1.0

КАК ПОЛЬЗОВАТЬСЯ:
1. Запустите файл main.py
2. На экране появится окно с кнопками
3. Нажимайте цифры, чтобы ввести число
4. Нажмите +, -, * или / для выбора действия
5. Нажмите =, чтобы получить результат
6. Нажмите C, чтобы очистить поле

ГОРЯЧИЕ КЛАВИШИ:
- ESC - закрыть программу
- Enter - посчитать результат

ЕСЛИ НЕ РАБОТАЕТ:
Убедитесь, что установлен Python.
